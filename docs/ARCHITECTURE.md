# EasyJob Architecture

## Project Overview
EasyJob is a comprehensive job management and application tracking platform that helps users manage job applications, track opportunities, and streamline the job search process.

## Directory Structure

### `/backend`
Node.js/Express API server
- **src/**: Source code for the backend application
- **tests/**: Test files for backend functionality
- **package.json**: Dependencies configuration

### `/frontend`
React web application
- **src/**: Source code for React components, pages, and utilities
- **public/**: Static assets (HTML, images, etc.)
- **package.json**: Dependencies configuration

### `/config`
Configuration files and environment templates
- **.env.example**: Template for environment variables

### `/docs`
Documentation files
- **ARCHITECTURE.md**: This file
- Other documentation as needed

### `/scripts`
Utility scripts for development, deployment, and automation

## Technology Stack

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB (configurable)
- **Authentication**: JWT

### Frontend
- **Framework**: React
- **Routing**: React Router
- **HTTP Client**: Axios
